## Meditation List!

### See it in action

Go to [http://xeniatay.com/meditation/](http://xeniatay.com/meditation/)!

To run locally: 

    // In the root directory of meditation-list
    > npm install
    > npm start

    Go to localhost:8080

### Project Requirements

- npm 4.0.3 
- Node v6.5.0 
- OSX

### Project Tools

- Webpack
- Babel
- ES6
- React
- SASS
- Bootstrap
